Interface IPriorityArbitrator

Namespace
EasySave.Models.Backup.Abstractions
Assembly
EasySave.dll

Defines a global priority arbitrator that manages file processing order across all backup jobs. Ensures that standard (non-priority) files are not processed while any priority files remain in the system, enforcing the v3 business rule globally.
