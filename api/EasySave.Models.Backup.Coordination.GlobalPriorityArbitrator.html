Class GlobalPriorityArbitrator

Namespace
EasySave.Models.Backup.Coordination
Assembly
EasySave.dll

Implements global priority arbitration across multiple backup jobs. Maintains a global counter of remaining priority files and prevents standard file processing while any priority files remain in the system, ensuring v3 business rule compliance.
